# เบอร์มงคล 手機號碼吉凶分析

輸入手機號碼，依《เปลี่ยนชีวิตด้วยพลังตัวเลข》(SIM SO SUCCESS) 書中邏輯評分，列出號碼帶給主人的優點與缺點。

## 使用

線上版：https://315112yjvs.github.io/sim-mongkol/ ，或直接用瀏覽器打開 `index.html`。
支援網址參數 `?n=號碼&d=出生星期`（1=週日…7=週六、8=週三夜）直接帶入分析。

## Chrome 插件（extension/）

自動掃描網頁上的泰國手機號碼（092-867-5576、0928675576、+66 等格式），在號碼旁顯示分數標籤，點標籤開啟完整分析頁。分數後帶「!」表示含書中大凶配對。彈出視窗可設定自動掃描開關與出生星期（塔克夏個人化分數）。

安裝：Chrome 開 `chrome://extensions` → 開右上角「開發人員模式」→「載入未封裝項目」→ 選 `extension` 資料夾。

## 規則摘要

- 只看後 8 碼（前 2 碼門號前綴不計），以相鄰兩碼配對為單位分析
- 配對分類：คู่มิตร 吉配、คู่สมพล 助力配、คู่ธาตุ 五行配、คู่ศัตรู 仇敵配
- 含仇敵配對的號碼依書中定義不算真正的吉祥號（結果會標註，但分數不設上限，該多少是多少）
- 三碼凶數組合（尤其含 0）另外扣分
- 選填出生星期可啟用塔克夏（ทักษา）個人化分析：八宮輪轉找出個人吉星與凶星數字，錢財、貴人 +2，權勢、資產 +1，凶星 -3，上限 ±12
