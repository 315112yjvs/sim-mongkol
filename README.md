# เบอร์มงคล 手機號碼吉凶分析

輸入手機號碼，依《เปลี่ยนชีวิตด้วยพลังตัวเลข》(SIM SO SUCCESS) 書中邏輯評分，列出號碼帶給主人的優點與缺點。

## 使用

直接用瀏覽器打開 `number-analyzer.html`，不需要伺服器。

## 規則摘要

- 只看後 8 碼（前 2 碼門號前綴不計），以相鄰兩碼配對為單位分析
- 配對分類：คู่มิตร 吉配、คู่สมพล 助力配、คู่ธาตุ 五行配、คู่ศัตรู 仇敵配
- 含仇敵配對的號碼依書中規則封頂 59 分
- 三碼凶數組合（尤其含 0）另外扣分
- 選填出生星期可啟用塔克夏（ทักษา）個人化分析：八宮輪轉找出個人吉星與凶星數字，錢財、貴人 +2，權勢、資產 +1，凶星 -3，上限 ±12；凶星扣分可穿透 59 封頂，吉星加分不能突破封頂
